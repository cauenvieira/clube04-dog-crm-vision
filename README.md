# CRM Clube04 — Mockup Visual

Mockup frontend completo e navegável do CRM de Leads do **Clube04 Mogi das Cruzes** (pet spa).

## ⚠️ Importante

Este é um **protótipo visual/UX**, não um sistema funcional completo.

- ❌ **Sem backend real** — dados ficam apenas em `localStorage`.
- ❌ Sem banco real, sem API real, sem WhatsApp real, sem IA real.
- ❌ Sem autenticação, sem integrações com n8n, ERP, Google ou Meta.
- ✅ **Funciona como referência visual** para implementação futura no CRM real.

Use este mockup como **referência de design, fluxo e nomenclatura** para o time de desenvolvimento adaptar no sistema definitivo.

## 🎯 Objetivo do produto

Substituir a planilha de **Jornada do Lead** com uma experiência SaaS moderna, centralizando cadastro, acompanhamento e histórico dos leads que chegam pelo WhatsApp, Instagram, Facebook, indicação, fachada e cadastros manuais.

## 📐 Módulos principais (prioridade alta)

- **Mesa Operacional** — tela principal do atendente, com Kanban e cards de operação.
- **Base de Leads** — planilha sistematizada com filtros e exportação CSV.
- **Análise da Liderança** — fluxo de revisão de casos sensíveis.
- **Dashboard de Leads** — KPIs, taxas e funil.
- **Resumo Diário** — fechamento operacional do dia.

## 🚀 Módulos secundários

- Nutrição e Campanhas
- Oportunidades

## 🌱 Módulos futuros (apenas visão)

- Biblioteca de Mensagens
- Biblioteca de Mídias
- Assistente IA
- Integrações reais (WhatsApp, Google Auth, automações)

## 🎨 Identidade visual

- Laranja como cor principal da marca Clube04.
- Fundo off-white acolhedor, sidebar escura sofisticada.
- Cards arredondados, chips de status discretos, ícones para leitura rápida.
- Visual desktop-first, responsivo básico para notebook.

## 💾 Persistência mockada

Dados ficam em `localStorage` (`clube04_leads_v1`). Em **Configurações → Modo demo** é possível reiniciar os dados mockados.

## 🗣️ Tom de linguagem

Português do Brasil, operacional. Usa "tutor", "doguinho", "atendimento", "liderança", "campanha" — evita termos técnicos.
